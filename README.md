# 🏟️ Club Management System (C++)

A professional command-line application built with **C++** to manage club members and activities. This project demonstrates modern C++ programming techniques and organized software design.

---

## 🚀 Key Features
- **Member Management:** Full CRUD operations (Add, Display, Search, Delete).
- **Data Persistence:** Integrated with file handling to ensure records are saved permanently.
- **Robust Logic:** Handles member data efficiently using object-oriented principles.

## 🛠️ Technical Implementation
In this project, I utilized several core C++ concepts to ensure code quality and efficiency:

- **Classes & Objects:** Used to model "Member" and "Club" entities, ensuring encapsulation and clean code.
- **Vectors (`std::vector`):** Employed for dynamic data storage, allowing the system to handle a flexible number of records without memory overflow.
- **Enums:** Used for defining fixed sets of options (like Member Status or Gender), which makes the code more readable and less prone to errors.
- **Structs:** Used for lightweight data grouping where appropriate.
- **File Handling (`fstream`):** Implemented to read from and write to external files, bridging the gap between RAM and disk storage.

## 💻 How to Run
1. Open the `.sln` file in **Visual Studio 2022**.
2. Build the solution (`Ctrl + Shift + B`).
3. Press `F5` to run the system.

## 📄 Educational Context
This project was developed as part of my studies in **Data Structures and File Handling**, focusing on how to manage data in memory and sync it with permanent storage.
